Middleware и обработка запросов
1️⃣ Как кратко ответить
Middleware (промежуточное ПО) — это компонент, который обрабатывает HTTP-запросы и ответы между клиентом и основным приложением. Middleware образует цепочку: каждый запрос проходит через все middleware до обработчика, а ответ — обратно. Middleware используется для аутентификации, логирования, CORS, сжатия, обработки ошибок, добавления заголовков и других сквозных задач, не загромождая бизнес-логику.

2️⃣ Подробное объяснение темы
Зачем нужен Middleware?
Middleware решает задачи:

Сквозная функциональность: Логика, общая для всех запросов
Разделение ответственности: Изоляция инфраструктурного кода
Цепочка обработки: Последовательная обработка запроса
Переиспользование: Одна реализация для всего приложения

Как работает Middleware:

Клиент → Middleware 1 → Middleware 2 → ... → Handler
                                                 ↓
Клиент ← Middleware 1 ← Middleware 2 ← ... ← Response

Middleware в FastAPI:

from fastapi import FastAPI, Request
from starlette.middleware.base import BaseHTTPMiddleware
import time
​
app = FastAPI()
​
# Способ 1: Через декоратор
@app.middleware("http")
async def add_process_time_header(request: Request, call_next):
    """Добавляет время обработки в заголовок ответа."""
    start_time = time.time()
    response = await call_next(request)
    process_time = time.time() - start_time
    response.headers["X-Process-Time"] = str(process_time)
    return response
​
# Способ 2: Через класс
class LoggingMiddleware(BaseHTTPMiddleware):
    """Логирование всех запросов."""
    
    async def dispatch(self, request: Request, call_next):
        logger.info(f"Запрос: {request.method} {request.url.path}")
        
        response = await call_next(request)
        
        logger.info(f"Ответ: {response.status_code}")
        return response
​
app.add_middleware(LoggingMiddleware)

Middleware для аутентификации:

from fastapi import Request, HTTPException
from starlette.middleware.base import BaseHTTPMiddleware
from jose import jwt, JWTError
​
class AuthMiddleware(BaseHTTPMiddleware):
    """Проверка JWT токена для защищённых маршрутов."""
    
    EXCLUDED_PATHS = ["/", "/docs", "/openapi.json", "/login", "/register"]
    
    async def dispatch(self, request: Request, call_next):
        # Пропускаем публичные маршруты
        if request.url.path in self.EXCLUDED_PATHS:
            return await call_next(request)
        
        # Проверяем токен
        auth_header = request.headers.get("Authorization")
        if not auth_header or not auth_header.startswith("Bearer "):
            return JSONResponse(
                status_code=401,
                content={"detail": "Missing authentication token"}
            )
        
        token = auth_header.split(" ")[1]
        try:
            payload = jwt.decode(token, SECRET_KEY, algorithms=[ALGORITHM])
            request.state.user_id = payload.get("sub")
        except JWTError:
            return JSONResponse(
                status_code=401,
                content={"detail": "Invalid token"}
            )
        
        return await call_next(request)
​
app.add_middleware(AuthMiddleware)

Rate Limiting Middleware:

from fastapi import Request
from starlette.middleware.base import BaseHTTPMiddleware
from starlette.responses import JSONResponse
import time
from collections import defaultdict
​
class RateLimitMiddleware(BaseHTTPMiddleware):
    """Ограничение количества запросов."""
    
    def __init__(self, app, requests_per_minute: int = 60):
        super().__init__(app)
        self.requests_per_minute = requests_per_minute
        self.requests = defaultdict(list)
    
    async def dispatch(self, request: Request, call_next):
        client_ip = request.client.host
        current_time = time.time()
        
        # Очищаем старые записи
        self.requests[client_ip] = [
            t for t in self.requests[client_ip]
            if current_time - t < 60
        ]
        
        # Проверяем лимит
        if len(self.requests[client_ip]) >= self.requests_per_minute:
            return JSONResponse(
                status_code=429,
                content={"detail": "Too many requests"},
                headers={"Retry-After": "60"}
            )
        
        self.requests[client_ip].append(current_time)
        return await call_next(request)
​
app.add_middleware(RateLimitMiddleware, requests_per_minute=100)

Request ID Middleware:

import uuid
from contextvars import ContextVar
​
request_id_ctx: ContextVar[str] = ContextVar('request_id')
​
class RequestIDMiddleware(BaseHTTPMiddleware):
    """Добавляет уникальный ID к каждому запросу."""
    
    async def dispatch(self, request: Request, call_next):
        request_id = request.headers.get("X-Request-ID") or str(uuid.uuid4())
        request_id_ctx.set(request_id)
        request.state.request_id = request_id
        
        response = await call_next(request)
        response.headers["X-Request-ID"] = request_id
        
        return response

Обработка ошибок:

from fastapi import Request
from fastapi.responses import JSONResponse
from starlette.middleware.base import BaseHTTPMiddleware
import traceback
​
class ErrorHandlerMiddleware(BaseHTTPMiddleware):
    """Глобальная обработка ошибок."""
    
    async def dispatch(self, request: Request, call_next):
        try:
            return await call_next(request)
        except Exception as exc:
            logger.error(
                f"Unhandled error: {exc}\n{traceback.format_exc()}"
            )
            
            return JSONResponse(
                status_code=500,
                content={
                    "detail": "Internal server error",
                    "request_id": getattr(request.state, 'request_id', None)
                }
            )

Порядок Middleware:
Порядок добавления важен — последний добавленный выполняется первым:

# Выполняется: ErrorHandler → RequestID → Auth → Logging → Handler
app.add_middleware(LoggingMiddleware)
app.add_middleware(AuthMiddleware)
app.add_middleware(RequestIDMiddleware)
app.add_middleware(ErrorHandlerMiddleware)

Middleware с зависимостями:

from fastapi import Depends
​
# Альтернатива middleware — зависимости
async def get_db():
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()
​
async def get_current_user(
    request: Request,
    db = Depends(get_db)
):
    user_id = request.state.user_id
    return db.query(User).get(user_id)
​
@app.get("/profile")
async def get_profile(user = Depends(get_current_user)):
    return user

Где применяется Middleware?

Аутентификация: Проверка токенов и сессий
Логирование: Запись всех запросов и ответов
CORS: Обработка кросс-доменных запросов
Rate Limiting: Защита от DDoS и перегрузки
Сжатие: GZip для уменьшения трафика
Кэширование: HTTP кэширование
Метрики: Сбор статистики запросов
Безопасность: Проверка заголовков, CSP
